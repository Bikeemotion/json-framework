### Adding Dependency
```
<dependency>
    <groupId>com.bikeemotion</groupId>
    <artifactId>json-framework</artifactId>
    <version>1.0.0</version>
</dependency>
```

# Testing it
#### Pre-requisites

* JDK 8 or newer
* Maven 3.1.0 or newer

#### Build
```
mvn clean install
```
